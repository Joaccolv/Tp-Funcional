!!Crear un repositorio!!
--git init [Inicializa un nuevo repositorio de git]
--git status [muestra por consola los archivos que todavia no subiste al repositorio]

!!Hacer un commit!!
--git add <archivo> [para agregar un aruchivo al commit]{si pones un . sube todos los archivos}
--git commit -m "Mensaje"[Mensaje corto para explicar que cambios hiciste en el repositorio]

!!Hacer un push!!
git push origin master [Envia los cambios realizados en el repositorio local al repositorio ubicado en github]

--git log [Te deja ver el historial de commits realizados]

!!Vincular con github!!
--git config --global user.email "Pones tu email"
--git config --global user.name "Nombre"

git clone "url" [clonar un repositorio de git]
